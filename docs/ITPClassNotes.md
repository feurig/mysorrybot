## How is OpenSCAD pronounced?
Open ESS Cad

## Where to get OpenScad
### Download binary
### install package
#### Ubuntu/Debian
```
root@redshirt:/home/feurig# apt install openscad
```
#### Macos / Homebrew
```
feurig@Nix ~ % brew install openscad
```
